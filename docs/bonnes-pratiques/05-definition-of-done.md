# Definition of Done et commits

## Definition de pret (DoD)

- Le besoin est compris et trace dans une issue ou une tache.
- Le code est relu et valide par au moins un autre developpeur.
- Les tests pertinents sont ajoutes et executables localement.
- Le comportement est documente dans le README ou la doc technique.

## Exemples de messages de commit

- `feat(intent): ajoute la detection des commandes meteo`
- `fix(prototype): gere correctement la commande d'arret`
- `docs(guide): enrichit les bonnes pratiques de codage`
- `test(simulation): ajoute un scenario de non-reconnaissance`
